About

Stencila Hub brings you Stencila with its all available feature without the need to install anything on your machine. All you need is an up-to-date browser and stable internet connection.
Stencila Hub provides you with the native Stencila editor and the standard Stencila execution engines for various programming languages. This means you can start your data analysis straight away. By setting up an account on Stencila Hub, you will be able to set up projects which you can share with your collaborators (as long as they have an account on the Hub), add data and analysis files from different sources and track the changes done to the project.
You can chose whether your projects on Stencila Hub are publicly visible or only accessible to those who collaborate on them. Public projects are a great way to turn your reports into interactive publications.
Learn more on how to use Stencila Hub.
